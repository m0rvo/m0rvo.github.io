Атрибут target

По умолчанию, при переходе по ссылке документ открывается в текущем окне или фрейме. Это поведение можно изменить с помощью атрибута target тега <a>. Этот атрибут может принимать следующие значения:

  • _blank — загружает страницу в новое окно браузера;
  • _self — загружает страницу в текущее окно (значение по умолчанию);
  • _parent — загружает страницу во фрейм-родитель;
  • _top — отменяет все фреймы и загружает страницу в полном окне браузера.

Пример 1. Открытие документа в новом окне

<!DOCTYPE html>
<html lang="ru">
 <head>
  <meta charset="UTF-8">
  <title>Новое окно</title>
 </head>
 <body>
  <p><a href="https://webref.ru" target="_blank">Ссылка открывает новое окно на сайт webref.ru</a></p> 
 </body>
</html>

Пример 2. Открытие документа во фрейме

<!DOCTYPE html>
<html lang="ru">
 <head>
  <meta charset="UTF-8">
  <title>Открытие файла во фрейме</title>
 </head>
 <body>
   <p><a href="https://webref.ru" target="newframe">Сайт webref.ru</a></p> 
 </body>
</html>

Ссылка в примере 2 ведет на сайт webref.ru, который открывается во фрейме с именем newframe. Если указано неизвестное значение для target, например, имя фрейма написано с ошибкой, ссылка откроется в новом окне.

Пример 3. Использование тега <base>

<!DOCTYPE html>
<html lang="ru">
 <head>
  <meta charset="UTF-8">
  <title>Открытие файла во фрейме</title>
  <base target="_blank">
 </head>
 <body>
  <p><a href="1.html">Ссылка откроется в новом окне</a></p>
  <p><a href="2.html" target="_self">Ссылка откроется в текущем окне</a></p>
 </body>
</html>

Чтобы все ссылки на странице открывались в новом окне, можно использовать тег <base target="_blank">. Для открытия ссылки в текущем окне можно добавить к тегу <a> атрибут target="_self".
