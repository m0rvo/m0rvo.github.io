Цвета ссылок в HTML

Цвета ссылок задаются с помощью атрибутов тега <body>. Если они не указаны, используются значения по умолчанию:

  • link — определяет цвет обычных ссылок (по умолчанию синий, #0000FF).
  • vlink — цвет уже посещенных ссылок (по умолчанию фиолетовый, #800080).
  • alink — цвет активной ссылки (по умолчанию красный, #FF0000).

Цвета могут задаваться в шестнадцатеричном формате или с использованием ключевых слов. В следующем примере цвет фона веб-страницы установлен черным, а цвет ссылок — красным.

Пример 1. Задание цветов ссылок

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
<html>
 <head>
  <meta http-equiv="Content-Type" content="text/html; charset=utf-8">
  <title>Цвет ссылок</title>
 </head>
 <body link="red" vlink="#cecece" alink="#ff0000" bgcolor="black">

  <p><a href="content.html">Содержание сайта</a></p>
 </body>
</html>
